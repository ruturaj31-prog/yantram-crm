# Yantram CRM - Lead Pipeline Management

A comprehensive WordPress CRM plugin for managing leads, pipelines, and sales workflows.

## Features

- 📊 Dashboard with lead statistics
- 👥 Lead management system
- 📈 Pipeline tracking (New, Active, Hold, A-Old, Close)
- 📋 Contact management
- ✅ Task management
- 📊 Reports and analytics
- 🔍 Advanced search and filtering
- 📱 Responsive design

## Installation

1. Download the plugin files
2. Upload to `/wp-content/plugins/yantram-crm/`
3. Activate the plugin in WordPress admin
4. Go to Yantram CRM menu to start managing leads

## Database Tables

- `wp_yantram_leads` - Lead information
- `wp_yantram_contacts` - Contact details
- `wp_yantram_tasks` - Task management
- `wp_yantram_activities` - Lead activities
- `wp_yantram_pipeline_stages` - Pipeline stages

## Usage

### Adding a Lead
1. Navigate to Yantram CRM > Leads
2. Click "+ Add Lead"
3. Fill in the lead details
4. Click "Add Lead"

### Managing Pipeline
- View leads organized by pipeline stages
- Drag and drop to change status
- Track lead progression

### Searching and Filtering
- Search by lead name, email, or phone
- Filter by status, work type, or source
- Export filtered results
